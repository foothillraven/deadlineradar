Iowa CPA License Renewal Deadline 2027

Iowa CPA license renewal

Iowa CPA/LPA Individual Certificate or License
Last verified 2026-07-16
Next renewal date
June 30, 2027Annual renewal -- every CPA certificate and LPA license (Iowa Code 542.6/542.8/542.19) expires June 30 each year, same date for every licensee. Iowa Admin. Code 193A-5.3(1): 'Licenses expire on June 30 of each year.' A renewal notice typically goes out in the May preceding expiration; a 30-day grace period follows a missed deadline before the license lapses (193A-5.4).
Iowa Admin. Code 193A-5.3(1)

CPA/LPA Firm Permit to Practice
Last verified 2026-07-13
Next renewal date
June 30, 2027Iowa Code § 542.7 establishes a firm permit to practice, separate from individual certificates/licenses. Firm-permit renewal is governed by Iowa Admin. Code 193A-Chapter 8 ('Licensed Public Accounting Firms'), not Chapter 5 (individual-license renewal only): permit holders must renew online with the Board by June 30 each year (rule 193A-8.3), with a 30-day grace period (25% late penalty) before the permit lapses on July 31 -- confirmed independently by the Board's own CPA Renewal FAQ.
Iowa Admin. Code r. 481-1028.2
